# Data-quality-checker-and-cleaner

A web app to check and clean your data instantly — no coding needed!

##  Live App
https://data-quality-checker-and-cleaner-yydkgrrqdh3eturfwyxvzl.streamlit.app/

##  Features
- Upload CSV or Excel file
- Auto-detects missing values & duplicates
- Data Quality Score out of 100
- Before vs After cleaning comparison
- Drop & rename columns
- Interactive charts — correlation, histogram, scatter plot, outlier detection
- Download cleaned data as CSV or Excel

##  Built With
- Python
- Streamlit
- Pandas
- Plotly
- VS Code

##  Run Locally
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py
