# Data-quality-checker-and-cleaner

A powerful web application to analyze, clean, and improve dataset quality — built using Python and Streamlit.
This tool helps users quickly identify missing values, duplicates, outliers, and overall data health — without writing any code.

##  Live App
https://data-quality-checker-and-cleaner-yydkgrrqdh3eturfwyxvzl.streamlit.app/

##  Features
- Upload CSV or Excel files
- Automatic detection of missing values & duplicates
- Data Quality Score (0–100)
- Before vs After cleaning comparison
- Drop & rename columns
- Interactive charts — correlation, histogram, scatter plot, outlier detection
- Download cleaned data as CSV or Excel

##  Built With
- Python
- Streamlit
- Pandas
- Plotly
- VS Code

##  Run Locally
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py
